The Document Object Model is a way of representing a page of HTML as a tree of nodes. The document.getElementById() , document.getElementsByClassName() , document.getElementsByTagNames() and document.querySelector() can be used to access elements on a page. The parentNode() , previousSibling() , nextSibling() , childNodes() and children() methods can be used to navigate around the DOM tree. The createElement() and createTextNode() methods can be used to create dynamic markup on the fly. Markup can be added to the page using the appendChild() and insertBefore() methods. Elements can be replaced using the replaceChild() method, and removed using the removeChild() method. The innerHTML property can be used to insert raw HTML directly into the DOM.
